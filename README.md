Requirements : 
Python 3.x
Tkinter (usually included with Python)
[Optional] Activate the virtual environment in myenv for package isolation.


How to Run
1.  Activate the virtual environment:

Windows:  myenv\Scripts\activate
Unix/macOS : source myenv/Scripts/activate


2 . Run any example:  python hello.py



Example Scripts :

hello.py: Displays a window with an image and sets a custom icon.
button.py: Shows a button and label; clicking the button updates the label.
frame.py: Demonstrates frames and button interaction.
grid.py: Uses grid layout for labels.
line.py: Draws lines on a canvas.
multilayout.py: Combines pack and grid layouts.
multiwidgetsgrid.py: Shows multiple widgets in a grid.
packlayout.py: Uses pack layout for labels.
placelayout.py: Uses place layout for labels.
scalewidgets.py: Interactive scale widget updates a label.
tempconv.py: Simple Fahrenheit to Celsius temperature converter.


Images
Place images (e.g., clear.png) in the tkinter/images/ directory..
